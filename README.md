# 📊 Credit Risk & Transaction Analysis (NBFC Domain)

## 🔍 Project Overview
This project focuses on exploratory data analysis (EDA) of transactional and customer data in the NBFC (Non-Banking Financial Company) domain to identify:

- Credit risk patterns  
- Fraudulent transaction behavior  
- Customer segmentation insights  
- Lending risk indicators  

The analysis is aligned with real-world financial services use cases, supporting data-driven lending and risk management decisions.

---

## 🎯 Business Objective

- Identify high-risk customers based on transaction behavior  
- Detect potential fraud patterns  
- Analyze customer spending and repayment trends  
- Support credit decisioning and underwriting strategy  
- Improve portfolio monitoring and risk control  

---

## 🧠 Key Insights & Outcomes

- Identified behavioral patterns linked to high default probability  
- Detected anomalies indicating potential fraudulent transactions  
- Segmented customers into low, medium, and high-risk categories  
- Provided actionable insights to support credit policy optimization  

---

## 🛠️ Tech Stack

- Python (Pandas, NumPy)
